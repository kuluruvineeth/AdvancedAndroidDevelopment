# Room Crud Application
A simple crud operations application yet powerful making use of MVVM Pattern and all advanced architectural components.

## Libraries/Modules/Concepts covered while building this application
1. MVVM Pattern
2. Room
3. RecyclerView
4. ViewModel
5. LiveData
6. Coroutines
7. Events
